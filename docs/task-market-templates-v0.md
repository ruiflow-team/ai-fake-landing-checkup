## 1. 模板包目录 v0

### Template 1｜AI 假落地体检评分表
- 用途：判断 AI 项目是否值得继续投入。
- 交付物：Markdown / 表格 / 评分说明。
- 可售路径：免费版引流，深诊断付费。

### Template 2｜7 天一人公司验证计划
- 用途：把想法拆成 7 天执行动作。
- 交付物：Day1-Day7 checklist。
- 可售路径：训练营/陪跑。

### Template 3｜AI 工具成本/隐私自查表
- 用途：检查是否上传敏感资料、是否成本不可控。
- 交付物：风险清单 + 替代方案。
- 可售路径：企业/小团队体检。

### Template 4｜真实用户评测任务发布模板
- 用途：合规招募真实反馈，不刷量、不诱导好评。
- 交付物：任务发布表 + 用户反馈表。
- 可售路径：任务托管服务。

### Template 5｜AI 内容质量门检查表
- 用途：AI 生成内容外发前检查质量和风险。
- 交付物：发布前 10 项检查。
- 可售路径：内容团队/个人 IP SOP。

## 2. 任务市场 v0 类型

### Task A｜5 人真实体验反馈
- 输入：公开产品链接、测试目标、反馈模板。
- 输出：原始反馈 + 高频问题 Top5 + 改进建议。
- 红线：不要求好评、不刷量、不虚假下载。

### Task B｜AI 假落地快速诊断
- 输入：项目一句话、工具栈、卡点。
- 输出：风险评分 + 最小行动建议。
- 红线：不承诺收益。

### Task C｜AI 工作流 SOP 改造
- 输入：现有人工流程。
- 输出：可复用 SOP + AI 节点 + 人工质量门。
- 红线：不上传密钥/隐私。

### Task D｜Landing/内容钩子体检
- 输入：landing 页/文章/短视频脚本。
- 输出：钩子评分、风险声明、CTA 修改建议。
- 红线：不夸大收益，不伪造案例。
