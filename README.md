# AI 假落地体检清单 v0

很多 AI 项目看起来很厉害，最后却没有真实用户、没有交付、没有收入。

这个仓库提供一套免费自检工具，帮助个人、一人公司和小团队判断：

```text
你的 AI 项目是真落地，还是假落地？
```

## 你可以得到什么

- `AI 假落地 25 问`：快速检查项目风险
- `100 分评分表`：判断是否值得继续投入
- `样刊 #000`：为什么很多 AI 项目看起来厉害却赚不到钱
- `Issue 体检入口`：你可以提交一个公开、非隐私的项目描述，获取基础风险提示

## 重要声明

- 不承诺收益
- 不提供投资/法律/财务建议
- 不鼓励刷量、诱导好评、虚假评论或绕平台规则
- 不要提交密钥、源码、客户隐私、商业机密或任何敏感资料
